The Game

How it works

01

Create a crew

One player opens the link and taps Create a Crew — they get a 4-letter room code to share.

02

Friends join

Everyone joins from their own phone with the code. 2–8 players per table.

03

Fill seats with bots

Short a few players? Drop in Easy, Medium, or Hard AI bots straight from the lobby.

04

Bid & sail

Predict exactly how many tricks you'll take each round — nail it for points, miss and pay.

☠️

The Hard bot plays to win

It runs Information Set Monte Carlo Tree Search — sampling hundreds of hidden-hand worlds consistent with everything it has observed (~400 worlds per bid), then running a UCT tree search rolled out to the end of each round. No GPU, no training data — just search.
